Queue Order

This tool is developed as per the reformed queue order. It will give the queue for next day based on the present day last queue.

Instructions (Short Version):

  1. Click on Browse and select the Wordpad (.docx) file (for Windows users) or XYZ file (for MAC users) containing the last queue of present day.
  2. Then enter the baseline value in the text box and click on Submit.
